# MathSquare

A comprehensive math quiz application for teachers and students built with Android.

## Features

### Teacher Features
- Create and manage quizzes with custom questions
- Schedule quizzes with start and end dates/times
- Organize students by grade and section
- View student scores and quiz history
- Manage student numbers for registration
- Real-time Firebase integration

### Student Features
- Take quizzes assigned by teachers
- View quiz history with attempt tracking
- Practice with CSV-based quiz questions
- Track scores and progress

## Technology Stack
- Android (Java)
- Firebase Firestore (Real-time database)
- Material Design Components

## Setup Instructions

1. Clone the repository
2. Open in Android Studio
3. Configure Firebase project
4. Build and run
